Simple

UploadIt is not like other file sharing services you have used. We do file uploading and file downloading with no fluff in between. Our main priorities are the user, performance, and speed.

Secure

We want to make sure that file you are downloading is the file you expect. This service will likely be used to host malware traps for unsuspecting users. If you find a sketchy file, please report it.
